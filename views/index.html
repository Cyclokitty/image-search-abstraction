Cyclokitty's Image Search Machine

In the address bar above, change the search term to whatever you like and get back a bunch of image links and information about each of those links.

http://ck-image-search-machine.herokuapp.com/api/imagesearch/skyrim?offset=10

You can also limit the number of images you get back by changeing the offset number in the address above:

http://ck-image-search-machine.herokuapp.com/api/imagesearch/skyrim?offset=3

Curious about the search terms others have made in the past? http://ck-image-search-machine.herokuapp.com/api/imagesearch And see what others have searched.
